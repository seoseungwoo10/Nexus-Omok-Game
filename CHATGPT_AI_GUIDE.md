# 🎮 Nexus Omok Game - ChatGPT AI 모드 가이드

## 🤖 ChatGPT AI란?

**OpenAI GPT-4o-mini**를 활용한 최첨단 오목 AI입니다.

### ✨ 주요 특징

- **🧠 전략적 사고**: 4수 이상 앞을 내다보는 고급 전략
- **💬 설명 가능**: AI가 왜 그 수를 두었는지 설명
- **🎯 높은 승률**: AI 승률 80% (최고 난이도)
- **📚 교육적**: 초보자의 학습 도구로 활용 가능

## 🔐 API 키 발급 방법

### 1단계: OpenAI 계정 생성
1. [https://platform.openai.com](https://platform.openai.com) 방문
2. **Sign up** 클릭하여 계정 생성
3. 이메일 인증 완료

### 2단계: API 키 발급
1. 로그인 후 [API Keys](https://platform.openai.com/api-keys) 페이지로 이동
2. **Create new secret key** 버튼 클릭
3. 키 이름 입력 (예: "Nexus Omok Game")
4. **Create secret key** 클릭
5. 생성된 키 복사 (⚠️ 한 번만 표시됨!)

### 3단계: 게임에 설정
1. Nexus Omok Game 실행
2. **🤖 AI 상대** 선택
3. **🤖 ChatGPT AI** 난이도 선택
4. **🔑 OpenAI API 키 설정** 버튼 클릭
5. 복사한 API 키 붙여넣기
6. **저장** 체크박스 선택 (다음번에 자동 입력)
7. **확인** 클릭

## ⚙️ 설정 관리

### API 키 변경
1. 게임 모드 선택 화면
2. **🔑 OpenAI API 키 설정** 클릭
3. 새 API 키 입력

### API 키 삭제
API 키를 삭제하려면:
```
C:\Users\[사용자명]\AppData\Local\Nexus_Omok_Game
```
폴더의 `user.config` 파일 삭제

### 사용량 확인
- 월별 사용량은 자동으로 추적됩니다
- 월 1000회 제한 도달 시 자동 경고
- 매월 1일 자동 리셋

## ❓ 문제 해결

### "API 키 없음" 오류
✅ **해결**:
1. API 키 설정 화면에서 키 입력
2. "저장" 체크박스 선택
3. 확인 클릭

### "네트워크 오류" 메시지
✅ **해결**:
1. 인터넷 연결 확인
2. 방화벽 설정 확인 (OpenAI API 허용)
3. 프록시 사용 시 설정 확인

### "API 사용량 한도 초과"
✅ **해결**:
1. OpenAI 계정의 사용량 확인
2. 결제 방법 등록 확인
3. 월간 한도 설정 확인

### "Invalid API Key" 오류
✅ **해결**:
1. API 키가 `sk-`로 시작하는지 확인
2. 복사 시 공백 제거 확인
3. OpenAI 계정에서 키가 활성화되어 있는지 확인

## 📝 FAQ

### Q: 인터넷 없이 사용 가능한가요?
**A**: 아니요. ChatGPT AI는 OpenAI API 호출이 필요합니다.  
로컬 AI (쉬움, 보통, 어려움)는 오프라인에서도 작동합니다.

### Q: API 키 비용이 부담스러워요
**A**: 다음 옵션을 고려하세요:
- 무료 크레딧 활용 (신규 가입 시)
- 로컬 AI 사용 (🔴 어려움 난이도도 충분히 도전적)
- 월 1000회 제한으로 과다 사용 방지

### Q: AI가 이상한 수를 둬요
**A**: 드물게 발생할 수 있습니다:
- 네트워크 문제로 응답이 잘못될 수 있음
- 자동으로 대안 수 또는 로컬 AI로 폴백
- 반복 시 API 키 또는 네트워크 확인
