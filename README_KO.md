# 🎮 넥서스 오목 게임

.NET 8.0과 WPF로 제작된 Windows 데스크톱용 클래식 오목(Gomoku) 게임입니다.

![.NET Version](https://img.shields.io/badge/.NET-8.0-512BD4)
![Platform](https://img.shields.io/badge/Platform-Windows-0078D4)
![License](https://img.shields.io/badge/License-MIT-green)

## 📖 개요

**넥서스 오목 게임**은 15x15 격자판에서 두 명의 플레이어가 번갈아가며 흑돌과 백돌을 놓는 전통적인 보드 게임입니다. 가로, 세로 또는 대각선으로 정확히 5개의 돌을 먼저 연속으로 놓는 사람이 승리합니다.

## ✨ 주요 기능

### 핵심 게임플레이
- **15x15 표준 바둑판** - 명확한 격자선이 있는 클래식 오목판 크기
- **2인 로컬 모드** - 같은 PC에서 친구나 가족과 플레이
- **턴 기반 시스템** - 흑돌이 항상 먼저 시작, 자동 턴 전환
- **승리 감지** - 정확히 5개의 돌이 일렬로 놓였을 때 자동 감지
- **무승부 감지** - 승자 없이 바둑판이 가득 찼을 때 인식
- **승리 시각화** - 승리한 5개의 돌을 빨간색 선으로 강조

### 사용자 인터페이스
- **호버 미리보기** - 유효한 위치에 마우스를 올리면 반투명 돌 미리보기
- **현재 턴 표시** - 누구의 차례인지 명확하게 표시 (⚫ 흑 / ⚪ 백)
- **상태 메시지** - 하단에 실시간 게임 상태 업데이트
- **새 게임 버튼** - 언제든지 쉽게 게임 재시작
- **반응형 디자인** - 창 크기 조절 시 적절한 비율 유지

### 시각 효과
- **그라디언트 돌** - 흑돌과 백돌에 3D 같은 효과
- **화점(천원점)** - 주요 교차점에 전통 바둑판 마커 표시
- **사운드 피드백** - 돌을 놓을 때 시스템 비프음
- **부드러운 애니메이션** - 전문적인 UI 전환 효과

## 🎯 게임 규칙

1. **바둑판 크기**: 15x15 격자 (225개의 교차점)
2. **선공 플레이어**: 흑돌이 항상 먼저 시작
3. **승리 조건**: 정확히 5개의 연속된 돌
   - 가로, 세로, 대각선 정렬
   - **6개 이상의 돌은 승리로 인정되지 않음** (육목 금지)
4. **간소화된 규칙**: 금수 없음 (쌍삼, 쌍사 금지 룰 미적용)
5. **무승부**: 바둑판이 완전히 채워지면 무승부로 종료

## 🚀 시작하기

### 필수 요구사항

- **Windows 10/11** (64비트 권장)
- **.NET 8.0 SDK** 또는 Runtime
- **Visual Studio 2022** (개발용)
  - '.NET 데스크톱 개발' 워크로드 필요

### 설치 방법

#### 옵션 1: 릴리스 다운로드 (출시 예정)
1. [Releases](../../releases) 페이지에서 최신 릴리스 다운로드
2. ZIP 파일 압축 해제
3. `Nexus-Omok-Game.exe` 실행

#### 옵션 2: 소스에서 빌드
```bash
# 저장소 복제
git clone https://github.com/yourusername/Nexus-Omok-Game.git
cd Nexus-Omok-Game

# 프로젝트 빌드
dotnet build

# 애플리케이션 실행
dotnet run
```

#### 옵션 3: Visual Studio 사용
1. Visual Studio 2022에서 `Nexus-Omok-Game.sln` 열기
2. `F5` 키를 누르거나 **시작** 버튼을 클릭하여 빌드 및 실행

## 🎮 게임 방법

1. **게임 실행** - 애플리케이션 시작
2. **돌 놓기**
   - 빈 교차점 위에 마우스를 올리기
   - 돌이 놓일 위치 미리보기 확인
   - 클릭하여 돌 놓기
3. **게임 승리** - 상대보다 먼저 정확히 5개의 돌을 일렬로 배치
4. **다시 시작** - **🔄 새 게임** 버튼을 클릭하여 다시 플레이

### 조작법

| 동작 | 컨트롤 |
|------|--------|
| 돌 놓기 | 마우스 왼쪽 클릭 |
| 위치 미리보기 | 마우스 호버 |
| 새 게임 | "🔄 새 게임" 버튼 클릭 |
| 재시작 확인 | 예/아니오 대화상자 |

## 🎨 UI 구성요소

### 메인 윈도우 레이아웃
- **헤더** (상단)
  - 현재 턴 표시기
  - 새 게임 버튼
- **게임 보드** (중앙)
  - 15x15 격자 캔버스
  - 인터랙티브 돌 배치
  - 호버 효과

  - 게임 상태 메시지

### 색상 구성
- **바둑판 배경**: 밀색/황갈색 (#DEB887)
- **바둑판 테두리**: 새들 브라운 (#8B4513)
- **격자선**: 검정색
- **흑돌**: 진한 회색에서 검정색으로 그라디언트
- **백돌**: 흰색에서 연한 회색으로 그라디언트
- **승리선**: 빨간색 (70% 불투명도)
